# Coursera Dump

The script collecting data about 20 random courses from Coursera.org and save it as excel file.

# How to launch?

Before, install all dependencies:
- ```bash pip install -r requirements.txt```

Launch the script:
- ```python3 coursera.py file.xlsx```
